# ExportXiamiList
导出虾米歌单方便导入到网易云音乐

- 依赖
beautifulsoup4

- EXE 下载

http://pan.baidu.com/s/1i3pELUH （提取码：pvai）

- 使用方法
    1. 进入虾米网首页，点击顶栏导航上的「我的音乐」；
    2. 点击「音乐库」下面的 Tab 「收藏的歌曲」，然后复制此网页的链接；
    3. 运行程序，复制此网页链接到程序中，然后点击导出等待完成；
    4. 进入网易云音乐首页，将鼠标移到头像可以看到「导入歌单」，然后点击；
    5. 将 3 中得到的 xiami.kgl 上传，等待导入完成
